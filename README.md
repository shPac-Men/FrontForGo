# Разработка интернет-приложений (РИП) 2025

## Расчет энергопотребления различных устройств в квартире

Система предназначена для расчета энергопотребления для различных устройств. Гости могут просматривать каталог устройств и изучать подробную информацию о каждом устройстве. Инженеры-проектировщики могут создавать заявки на расчет энергопотребления, выбирая интересующие их устройства и указывая температуру и количество жителей. Консультанты могут обрабатывать заявки на расчеты.
