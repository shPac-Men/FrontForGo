# Разработка интернет-приложений (РИП) 2025

## Прогноз pH раствора

Система предназначена для расчета pH растворов. Гости могут просматривать элементы и изучать подробную информацию о каждом элементе. Исследователи могут создавать заявки на расчет pH, выбирая интересующие их элементы и указывая объем воды которым разбавится раствор. Сотрудники химической лаборатории могут обрабатывать заявки на расчеты.
